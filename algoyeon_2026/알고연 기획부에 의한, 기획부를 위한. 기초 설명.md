Q : 프로젝트하면 보통 어느정도 걸리나요?
A : 작년 시간 기준, 체감상 1~2주면 베이스 까진 끝나요. 
나중엔 보통 오류수정이나 로직 등을 바꾸긴 합니다.

Q : 이것만은 필수로 알아야된다 있을까요?
A : 강요는 아니지만 Python 기초정도는 알아두시는걸 매우 추천드립니다. 
작년 기준으로 Python 언어가 가장 많이 쓰였던거같아요. 
Python 언어가 직관적이고 구조가 쉬운것도 있고, 1학년분들은 아마 정보시간에 배우게 될거라서
위에 언급한 강의 들어가셔서 기초강의정도는 들으시는것을 추천드립니다.

Q : 알고연은 학교 시간 외에 추가 근무 있나요?
A : 행사 (ex:축전) 나 학교 대회 ( 작년같은경우 자동차 대회 ) 때 프로젝트 진행과 
의견조율을 위해 야자시간때 하는거 빼고는 거의 없습니다. 
사실상 추가근무같은 경우는 프로젝트 팀 내에서 조율해야 되는거라서, 
일단 알고연 전체 내에서에 야자시간에 나와서 하는것 빼고는 없습니다.
